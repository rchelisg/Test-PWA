# Battery Run Time Calculator - UI Design Summary

## 整体布局
- **禁止左右滑动**：通过 `overflow: hidden` 和 `user-scalable=no` 实现
- **响应式设计**：自动适配苹果手机14到17的所有屏幕尺寸
- **垂直布局**：标题区 → 按键区 → 子屏幕区 → 注脚区
- **子屏幕最大化**：占据所有剩余空间，无留空

## 标题区
- **背景颜色**：浅绿色 (#81C784)
- **文字**：白色，显示 "Battery Run Time Calculator"
- **字体**：系统默认字体，粗体
- **高度**：减少约25%（默认padding: 12px）
- **响应式调整**：
  - 小屏幕：字体12px，padding: 9px
  - 中等屏幕：字体14px，padding: 10.5px
  - 大屏幕：字体16px，padding: 12px

## 按键区
- **按键数量**：4个
- **按键文本**：Calc, DfTime, DfCost, DfLoad
- **样式**：无按钮样式，直接使用文字
- **当前页面指示**：蓝色下划线
- **高度**：显著缩减（默认padding: 8px 16px）
- **间距**：按键之间gap: 16px
- **响应式调整**：
  - 小屏幕：字体10px，padding: 4px 6px
  - 中等屏幕：字体12px，padding: 4px 8px
  - 大屏幕：字体14px，padding: 4px 8px

## 子屏幕区
- **背景颜色**：
  - Calc：浅红色 (#FFCDD2)
  - DfTime：浅蓝色 (#BBDEFB)
  - DfCost：浅黄色 (#FFF3E0)
  - DfLoad：浅灰色 (#E0E0E0)
- **默认显示**：Calc子屏幕
- **内容**：显示对应子屏幕名称
- **响应式调整**：
  - 小屏幕：字体16px
  - 中等屏幕：字体18px
  - 大屏幕：字体20px

## 注脚区
- **背景**：无底色
- **文字**：深灰色 (#666)，显示 "(c) 2026"
- **高度**：减少约30%（默认padding: 11px）
- **响应式调整**：
  - 小屏幕：字体9px，padding: 8.4px
  - 中等屏幕：字体10px，padding: 9.8px
  - 大屏幕：字体11px，padding: 11.2px

## 交互逻辑
1. **默认状态**：应用启动时显示Calc子屏幕，Calc按键显示蓝色下划线
2. **按键点击**：
   - 切换到对应子屏幕
   - 为当前按键添加蓝色下划线
   - 移除其他按键的下划线
   - 清空所有数据
3. **数据处理**：切换子屏幕时清空所有数据

## 响应式设计
- **小屏幕设备** (iPhone SE, iPhone 12 mini等)：最大宽度320px
- **中等屏幕设备** (iPhone 14, iPhone 15等)：宽度321px到375px
- **大屏幕设备** (iPhone 14 Plus, iPhone 15 Plus, iPhone 16 Pro Max等)：宽度376px到428px
- **平板设备和普通浏览器**：宽度429px及以上，最大宽度限制为428px，添加边框和阴影

## 技术实现要点
- **HTML结构**：语义化标签，清晰的层次结构
- **CSS**：Flexbox布局，媒体查询，响应式设计
- **JavaScript**：事件监听，DOM操作，类切换
- **PWA特性**：禁止左右滑动，响应式适配

## 设计目标
- **紧凑布局**：最小化标题区、按键区和注脚区的高度，最大化子屏幕区
- **清晰导航**：通过下划线明确指示当前页面
- **视觉区分**：不同子屏幕使用不同背景颜色，便于识别
- **响应式适配**：确保在所有苹果手机型号上显示正常
- **简洁美观**：去除不必要的装饰，保持界面干净整洁
